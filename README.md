# 2D Platformer Metroidvania System

## Project Overview

Study project for 2D Platformer Metroidvania game design using Unity.

Focuses on designing a maintainable and scalable system architecture.

This project conforms to the [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/).

## Getting Started

### Prerequisites

- **Unity**: Unity 6000.3.9f1 or higher.
- **Render Pipeline**: Universal Render Pipeline (URP)

### Key Packages

- **Input System (1.18.0)**
- **2D Tools**: Includes **2D Animation (13.0.4)** and **2D Sprite (1.0.0)** for VFX and character rendering.

### Installation

1. Clone the repository:
   `git clone https://github.com/zlzonkh/2D-Platformer-Metroidvania-System.git`
2. Open the project in Unity Hub.
3. Navigate to `Assets/Scenes/` and open **SampleScene.unity**.
4. Press **Play** to test the mechanics.
